[새 기능] 설정값 "직접 입력" 모드 (엑셀 안 열어도 됨)

프로그램 창 중앙의 "설정값 입력 방식" 에서 방식을 고를 수 있습니다.

  [엑셀 파일(설정.xlsx)]  : 기존과 동일하게 설정.xlsx 값을 사용 (기본값)
  [직접 입력]           : 창에 뜨는 칸에 값을 바로 입력 (설정.xlsx 불필요)

■ 직접 입력 사용법
  1) "직접 입력" 을 선택하면 표지 제목/강사명/수강인원/교육개요 입력칸이 나타납니다.
  2) 빈칸을 채우고, 로우데이터만 끌어다 놓은 뒤 [결과보고서 만들기 ▶] 클릭.

■ 두 방식 연동
  설정.xlsx 를 먼저 넣으면 그 값이 직접 입력칸에도 자동으로 채워집니다.
  → 엑셀로 불러온 뒤 창에서 일부만 수정하는 것도 가능합니다.

※ 문항 라벨은 로우데이터에서 자동 생성되므로 직접 입력 시 따로 적지 않아도 됩니다.
