# 교육 만족도 결과보고서 자동 생성기 v2.1 — 사용설명서 (드래그 앤 드롭 · EXE 배포판)

설문 **로우데이터(엑셀)** 와 **설정 파일(설정.xlsx)**, **템플릿 PPT** 3가지만 넣으면
결과보고서 PPT가 자동으로 완성됩니다. 그래프 수치·평균·응답률·주관식 의견은 모두 자동 계산됩니다.

> **이번 배포판의 달라진 점**
> - 파이썬을 몰라도 되는 **exe 실행파일**로 배포합니다.
> - 파일을 창에 **끌어다 놓기(드래그 앤 드롭)** 하면 종류(로우데이터/설정/템플릿)를
>   **자동으로 인식**합니다. 어느 칸에 넣을지 고민할 필요가 없습니다.
> - **설정.xlsx·템플릿이 exe 와 같은 폴더**에 있으면 자동으로 미리 채워집니다.
>   그때는 **로우데이터 한 개만** 끌어다 놓으면 됩니다.

---

## 0. 가장 빠른 사용법 (최종 사용자용)

1. **설정.xlsx** 를 열어 이번 교육에 맞는 값(표지·강사명·수강인원·교육개요)만 수정 → 저장.
2. **교육결과보고서_자동생성기_v3.exe** 더블클릭.
3. 창의 큰 파란 영역에 파일을 **끌어다 놓고** → **[결과보고서 만들기 ▶]** 클릭.
4. 완료되면 안내창이 뜨고, 저장 폴더를 바로 열어볼 수 있습니다.

> 더 짧은 안내는 `빠른시작.txt` 를 참고하세요.

## 1. 준비물 (입력 3종)

| # | 파일 | 설명 |
|---|------|------|
| ① | 설문 로우데이터 (`.xlsx`) | 구글폼 등에서 내려받은 **응답 원본**. 1행=문항 제목, 2행부터 응답 |
| ② | 설정 파일 (`설정.xlsx`) | 표지 제목·강사명·수강인원·교육개요 등 **교육마다 바뀌는 값** |
| ③ | 템플릿 (`템플릿_결보표양.pptx`) | 결과보고서 서식 PPT (건드리지 말고 그대로 사용) |

### 로우데이터 형식 (중요)
- **1행**: 문항 제목. `타임스탬프` 열은 자동으로 무시합니다.
- **객관식 문항**: 응답값이 **척도 라벨**인 열은 **자동으로 객관식으로 인식**되어
  분포·평균이 계산됩니다. **척도 종류를 자동으로 감지**하므로 아래 형식을 모두 지원합니다.
  - 5점 '그렇다'형: `매우 그렇다 / 그렇다 / 보통이다 / 그렇지 않다 / 매우 그렇지 않다`
  - 5점 '만족'형: `매우 만족 / 만족 / 보통 / 불만족 / 매우 불만족`
  - 5점 '우수/도움'형, **4점·7점 척도**도 자동 인식 (그래프 눈금이 자동으로 맞춰집니다)
- **주관식 문항**: 위 척도가 아닌 자유서술 열은 **주관식으로 인식**됩니다.
  제목에 `좋았던·장점·우수` 류가 있으면 "좋았던 점", `아쉬·개선·보완` 류가 있으면
  "아쉬웠던 점" 칸에 자동으로 들어갑니다.
- 문항 순서는 **번호(1., 2. …)** 를 기준으로 정렬됩니다.

> **문항 수는 자유입니다.** 템플릿(표준 7문항)보다 **적으면 남는 문항 슬라이드가 자동 삭제**되고,
> **많으면**(7개 초과) 초과분은 슬라이드가 없어 경고가 뜹니다(그때만 템플릿에 문항 슬라이드를
> 추가). 표준 서식은 **객관식 7 + 주관식 2** 이며, 여기에 맞추면 손댈 것이 가장 적습니다.

## 2. 설정.xlsx 작성법

파일을 열면 3개 시트가 있습니다. **노란 안내줄**과 **파란 머리글**은 그대로 두고,
값 칸만 교육에 맞게 고치면 됩니다.

### ① 기본정보 시트
| 항목 | 예시 값 | 설명 |
|------|---------|------|
| 표지_상단라벨 | 결과보고서 | 표지 짙은 박스 문구 |
| 표지_제목1 | SK TNS FLP 대상 | 표지 큰 제목 1줄 |
| 표지_제목2 | 인사이트 트립 결과보고서 | 표지 큰 제목 2줄 |
| 강사명 | 설상훈 교수 | 강사 문항·그래프에 자동 반영 |
| 수강인원 | 9 | 총 수강 인원 → **응답률 자동 계산** |

> 응답률(%) = 응답자 수 ÷ 수강인원 × 100. 응답자 수는 로우데이터에서 자동으로 셉니다.
> 저장 파일 이름은 표지 제목을 참고해 자동 제안되며, 창에서 바꿀 수 있습니다.

### ② 교육개요 시트
슬라이드 4의 "교육 개요" 표에 그대로 들어갑니다.
`과정명 / 교육일정 / 교육방식 / 교육대상 / 교육목표` 값만 채우세요.

### ③ 문항설정 시트 (선택 — 대개 손댈 필요 없음)
문항 **라벨**만 바꾸고 싶을 때 사용합니다. 문항 **내용**은 로우데이터에서 자동으로 가져옵니다.
- `섹션라벨(■)`: 각 문항 슬라이드 제목의 ■ 뒤 라벨
- `요약차트라벨`: 요약 그래프 가로축 라벨. `{강사}` 는 강사명으로 자동 치환,
  줄바꿈은 `\n` 으로 입력 (예: `{강사}\n만족도 1-1`)

## 3. 실행 방법

### 방법 A. EXE (권장 — 파이썬 불필요)
1. `교육결과보고서_자동생성기_v3.exe` 를 더블클릭합니다.
2. 창에 파일을 **끌어다 놓습니다**.
   - 큰 파란 영역에 **①·②·③ 세 파일을 한꺼번에** 끌어다 놓으면 자동 분류됩니다.
   - 아래 세 칸에 **하나씩** 끌어다 놓아도 되고, 칸을 **클릭해서 선택**해도 됩니다.
   - 설정·템플릿이 exe 옆에 있으면 이미 채워져 있으니 **로우데이터만** 놓으면 됩니다.
3. 세 칸이 모두 초록색 **"선택됨 ✓"** 인지 확인 → **④ 저장 위치** 확인(자동 지정됨).
4. **[결과보고서 만들기 ▶]** 클릭. 진행 로그가 흐르고, 끝나면 안내창이 뜹니다.

> 끌어다 놓기가 동작하지 않는 환경이라면(드래그 모듈 미탑재) 자동으로
> **"클릭해서 파일 선택"** 방식으로 바뀝니다. 사용법은 동일합니다.

### 방법 B. 파이썬으로 바로 실행 (개발/테스트용)
- `RUN.bat` 더블클릭(파이썬으로 창을 바로 띄움), 또는
- 명령어(CLI)로 창 없이 생성:
```bash
python report_generator.py ^
  --raw "설문_로우데이터.xlsx" ^
  --config "설정.xlsx" ^
  --template "템플릿_결보표양.pptx" ^
  --output "결과보고서.pptx"
```
옵션: `--keep-no-opinion` 을 붙이면 아쉬웠던 점의 "없음"류 응답도 그대로 표시합니다.
(기본값은 "없음/없습니다/해당없음" 등 무의미한 답변을 자동 제외)

## 4. 자동으로 처리되는 것

- 드롭한 파일의 **종류 자동 인식** (엑셀 시트/확장자로 판별)
- **척도 자동 감지** (5점 그렇다형·만족형·우수형·도움형, 4·7점 등)
  → 그래프 눈금·**막대 방향**·**축 라벨(매우 그렇다 등)** 이 척도에 맞게 자동 정렬
- **문항 수 자동 대응**: 문항이 템플릿보다 적으면 남는 문항 슬라이드 자동 삭제
- 문항별 **응답 분포 그래프**(가로 막대) 자동 계산
- 문항별 **평균 점수** 및 **요약 그래프**(세로 막대) 자동 계산
- 요약 그래프 라벨: 설정.xlsx `문항설정` 값 우선, 비워두면 **문항 제목에서 자동 생성**
- **응답률** 문구 (`수강인원 총 N명 중 M명 응답 ｜ 응답률 X%`) 자동 기입
- **교육 개요 표 / 설문 구성 표** 자동 기입 (강사명 자동 반영)
- **주관식 의견** 자동 정리 (좋았던 점 전체, 아쉬웠던 점은 무의미 답변 제외)
- 템플릿의 **안내용 파란 박스**("내용만 바뀜", "강사명 변경") 자동 삭제
- 주관식이 많아도 글자가 넘치지 않도록 **자동 크기 조정**
- 그래프의 **색상·서식**은 템플릿 그대로 보존됩니다.
- 생성 중에도 창이 멈추지 않도록 **백그라운드 처리**됩니다.

## 6. 자주 묻는 질문 / 문제 해결

- **드롭했는데 엉뚱한 칸에 들어가요** → 각 칸을 직접 클릭해 올바른 파일을 다시 선택하세요.
  (엑셀이 여러 개일 때, 파일명에 `설정` 이 있으면 설정으로, 아니면 로우데이터로 인식합니다.)
- **그래프 숫자가 이상해요 / 문항이 주관식으로 빠져요** → 응답값이 지원 척도
  (그렇다형·만족형·우수형·도움형, 4·5·7점)와 일치하는지 확인하세요. 사내에서 전혀 다른
  문구의 척도를 쓴다면 알려주시면 척도 목록에 추가해 드립니다.
- **강사명이 안 바뀌어요** → 설정.xlsx `기본정보`의 `강사명`을 확인하세요.
- **문항 수가 다른 교육** → **자동 대응합니다.** 표준(7)보다 적으면 남는 문항 슬라이드가
  자동 삭제됩니다. 7개를 **초과**하면 초과분 슬라이드가 없어 경고가 뜨는데, 그때만
  템플릿에 문항 슬라이드를 추가하세요. 라벨이 안 맞으면 설정.xlsx `문항설정`의 라벨을
  교육에 맞게 고치거나 라벨 칸을 비워 자동 생성되게 하세요.
- **응답률을 100%로 두고 싶어요** → `수강인원`을 응답자 수와 같게 입력하세요.
- **Windows 보안 경고(파란 화면)가 떠요** → 서명되지 않은 exe라 나올 수 있습니다.
  `추가 정보` → `실행` 을 누르면 됩니다. (사내 배포 시 정상)
- **exe 실행이 안 돼요** → `설정.xlsx`, `템플릿_결보표양.pptx` 를 exe와 **같은 폴더**에
  두었는지 확인하세요. 백신이 막는 경우 예외 등록이 필요할 수 있습니다.

## 7. (담당자용) 배포 파일(EXE) 만들기

파이썬이 설치된 PC에서 **`BUILD_EXE.bat` 를 더블클릭**하면 끝입니다.
`dist` 폴더 안에 아래가 만들어집니다.

- `교육결과보고서_자동생성기_v3.exe`  (프로그램 본체 — 파이썬 없어도 실행)
- `설정.xlsx`            (교육마다 이 파일만 수정)
- `템플릿_결보표양.pptx`  (서식, 그대로)
- `빠른시작.txt`, `사용설명서.md`

`dist` 폴더 내용을 통째로 복사해 팀원에게 전달하면 됩니다.

> 직접 명령으로 빌드하려면:
> ```bash
> pip install -r requirements.txt
> pyinstaller --onefile --noconsole --name "결과보고서생성기" ^
>   --add-data "assets;assets" --collect-all tkinterdnd2 --collect-data pptx ^
>   report_generator.py
> ```
> `--add-data`, `--collect-all tkinterdnd2` 는 **끌어다 놓기 기능과 기본 템플릿을
> exe 안에 함께 포함**하기 위한 것입니다(빠뜨리면 드래그가 동작하지 않습니다).

## 8. 변경 이력

### v2.1 (범용성·정합성 보강)
이전 버전에서 발견된 아래 항목을 자동으로 처리하도록 개선했습니다.

- **분포 그래프 상하 정합성 수정(중요).** 분포차트는 축이 숨겨져 있고 라벨이 별도
  텍스트박스라, 데이터 순서에 따라 막대가 위/아래로 뒤집힐 수 있었습니다. 이제
  **항상 '가장 높은 등급(예: 매우 그렇다)'이 맨 위 라벨과 맞도록** 데이터를 정렬합니다.
- **척도별 축 라벨 자동 교체.** 만족형·우수형·4·7점 등 다른 척도를 넣으면, 그래프
  옆의 라벨(매우 그렇다/그렇지 않다 …)도 해당 척도의 표현으로 **자동으로 바뀝니다.**
- **요약차트 세로축 자동 확장.** 7점 척도처럼 평균이 5를 넘는 경우에도 막대가 잘리지
  않도록 축 최댓값을 척도에 맞춰(예: 7점 → 8) 자동 조정합니다.
- **문항 라벨의 안전한 자동 생성.** `문항설정`의 라벨은 **객관식 문항 수가 실제와
  일치할 때만** 번호대로 적용합니다. 문항 수/구성이 다른 교육에서는 라벨을 **문항
  제목에서 자동 생성**하여, 엉뚱한 라벨(예: 교육운영 문항에 '강사 만족도')이 붙는
  일을 방지합니다. 깔끔한 라벨을 원하면 `문항설정`의 라벨 칸을 채우세요.
- **CSV 로우데이터 지원.** `.csv`(구글폼 CSV 내보내기 등)도 바로 읽습니다.
  (utf-8/cp949 자동 인식)
- **주관식 '없음' 필터 완화.** "없지만 매우 좋았습니다"처럼 뒤에 실제 내용이 이어지는
  긴 답변은 **더 이상 삭제하지 않습니다.** 짧고 사실상 내용이 없는 '없음/없습니다'류만
  제외합니다.

> 표준 형식(5점 '그렇다'형, 객관식 7 + 주관식 2)의 결과물은 이전과 동일하게 나오며,
> 위 개선은 주로 **다른 척도·다른 문항 구성**의 교육에서 효과가 있습니다.
